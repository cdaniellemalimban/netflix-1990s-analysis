# netflix-1990s-analysis
Analysis of Netflix movies from the 1990s using Python and pandas for data cleaning, aggregation, and exploratory analysis. <br>
Project imported from DataCamp, part of completed self-paced coursework. <br>

# Overview
This project explores Netflix movies released in the 1990s to uncover trends in genres, duration, and popularity.

# Objectives
Identify the most common genres and movie characteristics <br>
Analyze trends in movie duration and release year <br>
Extract insights relevant to nostalgic content production <br>

# Tools Used
Python, Pandas, Matplotlib

# Key Skills Demonstrated
Data cleaning and preprocessing <br>
Exploratory data analysis (EDA) <br>
Aggregation and ranking <br>
Data visualization
